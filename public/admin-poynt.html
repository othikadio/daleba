🔗 Option B — Webhook Natif GoDaddy Payments

Configurez votre URL webhook une fois dans le dashboard Poynt.
Chaque transaction arrivera en temps réel dans DALEBA.

1. Login → business.poynt.net → Settings
2. Webhooks → Add Webhook URL
Chargement URL…
3. Secret = valeur de GODADDY_WEBHOOK_SECRET
4. Events: TRANSACTION_CAPTURED, TRANSACTION_REFUNDED
5. Save → Test → vérifier statut ✅ dans les logs DALEBA
